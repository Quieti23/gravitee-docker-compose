# graviteeio api gateway docker-compose running

fork from graviteeio project && change some deps image

## how to run

```code
cd platform && docker-compose up -d
```

## manager ui

* api portal

```code
open https://localhost/apim/portal
```

* access manager ui

```code
open https://localhost/am/ui/
```
